Provides impl of command executors.

These are main executors

  • {@link org.vpda.common.core.command.executor.impl.CommandExecutorBase} is basic executor which just executes command in current thread
  • {@link org.vpda.common.core.command.executor.impl.CommandExecutorDelegate} delegates execution to another executor
  • {@link org.vpda.common.core.command.executor.impl.CommandExecutorThreadPool} executes commands in new thread using thread pool
  • {@link org.vpda.common.core.command.executor.impl.CommandExecutorVisitor} visits each command with passed visitor before each execution
