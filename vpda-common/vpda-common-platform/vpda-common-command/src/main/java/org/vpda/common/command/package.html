Provides classes and interfaces that encapsulates command pattern. Command pattern is widely used in framework as execution pattern also for communication.

These are main classes in this package

  • {@link org.vpda.common.core.command.Command} is executable element
  • {@link org.vpda.common.core.command.CommandExecutor} which executes command using passed environment
  • {@link org.vpda.common.core.command.CommandExecutionEnv} is environment from which commands can fetch references
