Provides main entry point contract to application and its modules. VPDA providers modular structure with one application entry point and module entry point for each module.

These are main interfaces in this package

  • {@link org.vpda.common.core.entrypoint.AppEntryPoint} is entry point to application. There will exists only one application entry point. Classes implementing this interface must initialize application and call entry point method of each module
  • {@link org.vpda.common.core.entrypoint.ModuleEntryPoint} is entry point to module. Its purpose is to initialize module
  • {@link org.vpda.common.core.entrypoint.AppConfiguration} is holder of application configuration
  • {@link org.vpda.common.core.entrypoint.ModuleConfiguration} is holder of module configuration
