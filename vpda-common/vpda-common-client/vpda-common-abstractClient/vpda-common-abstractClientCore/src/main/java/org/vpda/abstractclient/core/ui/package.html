Provides interfaces that encapsulates client user interface abstraction.

These are main classes in this package

  • {@link org.vpda.abstractclient.core.ui.Frame} is Frame abstraction implemented in real client platform
  • {@link org.vpda.abstractclient.core.ui.WindowManager} is manager of frmaes
  • {@link org.vpda.abstractclient.core.ui.ClientUI} encapsulates client user interface artifacts
  • {@link org.vpda.abstractclient.core.ui.LoginFrame} is abstraction to login frame
  • {@link org.vpda.abstractclient.core.ui.MainFrame} is abstraction for main application frame
  • {@link org.vpda.abstractclient.core.ui.FrameInfo} is structure used to create real client platform frames
