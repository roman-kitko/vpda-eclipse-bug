Provides client side of Providers. Define abstract user interface for all providers - its client side. Providers defined at server side using {@link org.vpda.clientserver.viewprovider.ViewProvider} are rendered at client side using {@link org.vpda.abstractclient.viewprovider.ui.ViewProviderUI} interface.

These are main interfaces in this package

  • {@link org.vpda.abstractclient.viewprovider.ui.ViewProviderUI} is abstraction for user interface of view provider
  • {@link org.vpda.abstractclient.viewprovider.ui.ViewProviderUIManager} manages ui providers for client application
  • {@link org.vpda.abstractclient.viewprovider.ui.ViewProviderUILayoutManagerFactory} creates layout managers
  • {@link org.vpda.abstractclient.viewprovider.ui.ViewProviderComponentsUIFactory} creates real platform components from provider components
