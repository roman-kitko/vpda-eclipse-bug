Provides client and server contract for view providers. View provider is typically defined on server implementing {@link org.vpda.clientserver.viewprovider.ViewProvider} interface and rendered on client.

These are main interfaces nad classes in this package

  • {@link org.vpda.clientserver.viewprovider.ViewProvider} is main contract of view provider
  • {@link org.vpda.clientserver.viewprovider.ViewProviderDef} is id definition of view provider. It is lookup key
  • {@link org.vpda.clientserver.viewprovider.ViewProviderInfo} is initialization info about ViewProvider
