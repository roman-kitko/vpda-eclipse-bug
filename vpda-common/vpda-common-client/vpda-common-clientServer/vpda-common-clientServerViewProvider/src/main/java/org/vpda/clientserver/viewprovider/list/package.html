Provides basic components used in list view providers user interfaces.

These are main interfaces nad classes in this package

  • {@link org.vpda.clientserver.viewprovider.list.ListViewProvider} is contract between client and server to define and render list view.
  • {@link org.vpda.clientserver.viewprovider.list.ColumnInfo} is definition of one column in list view.
  • {@link org.vpda.clientserver.viewprovider.list.ListViewProviderInfo} is information response from list view provider initialization.
